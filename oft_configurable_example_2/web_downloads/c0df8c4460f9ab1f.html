    Updated beta release of the Open FUSION Toolkit.

    Important

    This release introduces breaking changes for TokaMaker models. For models with poloidal field coils the coil definition dictionary (eg. from gs_Domain.get_coils()) must be passed to TokaMaker.setup_regions(). See the documentation for more information and the HBT-EP meshing and HBT-EP equilibrium examples for demonstrations.

    Note

    The beta status reflects possible issues that may have been introduced in the refactoring for open-sourcing. See Milestone #1 as a tracker for progress to a full v1.0.0 release.

    New features

    • TokaMaker: Add support for grouping coils into coil sets (TokaMaker: Add support for grouping coils into coil sets #38)
    • TokaMaker: Add function to compute loop voltage for plasma (Cleaned up loop voltage functionality, documentation, test functions #18)
    • TokaMaker: Expose Grad-Shafranov field interpolation functionality in Python (TokaMaker: Expose Grad-Shafranov field interpolation functionality in Python #34)
    • TokaMaker: Add support for inactive x-points outside the plasma region (TokaMaker: Add support for inactive x-points outside the plasma region #46)
    • TokaMaker: Add support for plasma row/column in mutual inductance matrix (TokaMaker: Add support for plasma row/column in mutual inductance matrix from Tok… #49)
    • TokaMaker: Add functionality to compute currents in conducting regions (TokaMaker: Add functionality to compute currents in conducting regions #47)
    • TokaMaker: Add support for partially overlapping curves in meshing (TokaMaker: Add support for partially overlapping curves in meshing #35)
    • TokaMaker: Add rotation flag to gs_Domain.add_rectangle() (Updated add_rectangle in Tokamaker.py #41)
    • TokaMaker: Add DIII-D example (Add DIII-D example for TokaMaker #48)
    • ThinCurr: Convert driver programs to use binary output format for history files (Convert ThinCurr to use binary output format for history files #26)
    • TokaMaker/Marklin: Ensure proper memory layout for arrays passed through Python wrapper (Ensure proper memory layout for arrays passed through Python wrapper #44)
    • Add support for code coverage and MPI integration tests (Add support for code coverage and MPI CI tests #33)
    • Add continuous delivery builds for commits to main (Add continuous delivery builds for commits to main #39)
    • Add website CI build with artifacts for pull requests (Add website CI build with artifacts for pull requests #36)
    • Add Python interface to code coverage (Add Python interface to code coverage #50)

    Fixes

    • ThinCurr: Correct resistivity units on filament coils to SI, fixes ThinCurr: Filament resistivity uses incorrect units #30 (ThinCurr: Correct resistivity units on filament coils to SI, fixes #30 #31)
    • TokaMaker: Fix operation with multiple time-dependent solves in a single Python kernel (TokaMaker: Allocation error with multiple calls to TokaMaker.setup_td() #42)
    • Add linker fix to build_libs for macOS 13 (Ventura), ref Building on macOS may fail with Xcode 15 and newer #12
